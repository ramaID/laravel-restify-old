## Installation

You can install the package via composer:

```bash
composer require codeweaver/laravel-restify
```

## Usage

### Setup requirements

```bash
php artisan restify:setup
```

### Generate data object and controller class

```bash
php artisan restify:gen {name}
```

## Testing

